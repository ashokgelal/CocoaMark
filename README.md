# CocoaMark
Simple Markdown rendering for Cocoa using hoedown under the hood

## Basic usage

```objective-c
#import "CocoaMark/CocoaMark.h"

NSString *htmlOutput = [CocoaMark renderMarkdown:@"Hello *world*!"];
```
